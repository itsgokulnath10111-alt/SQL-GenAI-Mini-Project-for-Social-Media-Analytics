# SQL + GenAI Mini Project: Social Media Analytics

## Project Overview
This project analyzes social media user behavior and engagement using SQL.
Generative AI was used as a supporting tool for query formulation,
optimization, and analytical insight generation.

## Technologies Used
- SQL (MySQL compatible)
- Generative AI (analytical assistance)
- GitHub

## Project Objectives
- Identify the most active users based on posts and comments
- Analyze popular posts, hashtags, and influencers
- Measure engagement trends across countries
- Detect inactive and passive users

## Project Files
- SQL-GenAI-Social-Media-Analytics-Mini-Project.pdf – Complete project report
- Social_Media_Analytics.sql – SQL queries and database logic

## How to Use
1. Create a database named `social_media`
2. Import the dataset tables
3. Execute the SQL queries step by step
4. Refer to the PDF for explanations and results
